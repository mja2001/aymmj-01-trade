## ⚙️ Installation

### Prerequisites
- Python 3.10+ (for async trading features)
- Git
- Docker (optional, for containerized backtesting)

### Clone and Setup
```bash
git clone https://github.com/<YOUR_ORG>/<YOUR_REPO>.git
cd <YOUR_REPO>

# Create virtual environment
python -m venv .venv
source .venv/bin/activate  # Linux/Mac
# .venv\Scripts\activate     # Windows

# Install dependencies
pip install -r requirements.txt
pip install -r requirements-dev.txt

# Copy env template
cp .env.example .env
# Edit .env with your API keys (e.g., ALPHAVANTAGE_API_KEY)
